Quill runtime authorization · decision plane

One deterministic decision before every action

Every tool call an AI agent makes runs through one local decision plane: a gate that decides, a kernel floor that enforces what the gate can't see, and a tamper-evident log that proves what happened. No model sits in the decision path, so the gate itself cannot be talked out of a verdict.

Agent action

Tool call

Bash · Edit · Write · MCP tool — anything the coding agent (Claude Code, Cursor) tries to execute.

Enforcement point

The Quill gate

A PreToolUse hook that fires before the call executes and holds it until a verdict comes back. The agent never reaches the tool unsupervised.

Decision point

Deterministic classifier

Compiled regex and policy, no LLM in the path, so it can't be prompt-injected or jailbroken. It weighs three signals at once:

risk classification · low → critical secret scan on writes lethal-trifecta taint
+ Allow
Runs. Logged silently.
? Ask
Operator approves — Touch ID where available.
✕ Deny
Blocked. The irreversible line never softens.
$ Redact
Runs with detected secrets stripped.
Kernel floor · enforces below the interpreter

Seatbelt sandbox

Whatever runs is confined by the OS on the real syscall: file writes are held to the project allowlist and network egress is sealed, inherited by every interpreter and subprocess the agent spawns. This catches what a text classifier structurally cannot — the script that writes its own payload, the raw requests.post exfil, the edit aimed at the gate's own config.

Evidence plane

Tamper-evident audit log

Every attempt — allowed, asked, denied, or redacted — is written to an HMAC-chained log and rolled up into an Agent Receipt: what it did, what changed, what it was uncertain about, what to verify.

The gate decides. The kernel floor enforces what the gate can't see. The log proves what happened. The honest claim is no known bypass for a non-root agent short of an OS-level zero-day or a misconfigured profile of our own — defense-in-depth, not a guarantee.
